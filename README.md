# text-mining---uncertainty_index
This repository contains R-codes to measure uncertainty using the budget speech in Malaysia 
It includes four index, namely average sentiment index (ASI), economic policy uncertainty, 
net-sentiment index and market sentiment index
1 R-codes to construct the average sentiment index (ASI) using Afinn lexicon
Basic formula for AUI = (summation of positive score)+(summation of negative scorces)/(total number of words or token)
2 R-codes to construct the economic policy uncertainty (EPU) index using the specific lexicon
Basic forumate for EPU = [uncertainty/(economy+policy)}x100
3 R-codes to construct the net sentiment index (NSI) and market sentiment index (MSI)
Basic formula for NSI = [(positive score - negative score)/(positive score + negative score)]x100
Basic formula for MSI = [(anticipation score - fear score)/(anticipation score + fear score)] x100
